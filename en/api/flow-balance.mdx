title: "FLOW Balance API"
description: "Query balance, breakdown by source, ledger entries."
---

## GET /me/flow-balance

Return the current user's FLOW balance, broken down by source. **Auth**: required.

```bash
curl https://api.agentflow.website/me/flow-balance \
  -H "Cookie: af_session=..."
```

**Response**

```json
{
  "balance": 412.6,
  "withdrawableFlow": 283.2,
  "breakdown": {
    "topUp": 50,
    "subscriptionGrant": 73.4,
    "creatorShare": 180,
    "holderRewardsTotal": 78.2,
    "freelance": 25,
    "quests": 6
  },
  "recentLedger": [
    {
      "id": "led_01HQ...",
      "delta": -0.5,
      "reason": "call",
      "refType": "agent_call",
      "refId": "call_01HQ...",
      "createdAt": "2026-04-25T11:42:00Z"
    }
  ]
}
```

<ResponseField name="balance" type="number" required>
  Current spendable FLOW balance.
</ResponseField>

<ResponseField name="withdrawableFlow" type="number" required>
  Subset of `balance` eligible for payout (excludes grant FLOW and unsettled top-ups).
</ResponseField>

<ResponseField name="breakdown" type="object" required>
  Lifetime totals by source. Useful for analytics.
</ResponseField>

<ResponseField name="recentLedger" type="object[]">
  Last 20 ledger entries. Use `ledger` endpoint for paginated history.
</ResponseField>

## GET /ledger

Paginated ledger history for the current user. **Auth**: required.

**Query params**

<ParamField path="cursor" type="string">
  Pagination cursor.
</ParamField>

<ParamField path="reason" type="string">
  Filter by reason: `topup`, `call`, `grant`, `payout`, `refund`, `quest`.
</ParamField>

<ParamField path="from" type="string">
  ISO 8601 timestamp.
</ParamField>

<ParamField path="to" type="string">
  ISO 8601 timestamp.
</ParamField>

```bash
curl "https://api.agentflow.website/ledger?reason=call&from=2026-04-01" \
  -H "Cookie: af_session=..."
```

**Response**

```json
{
  "items": [
    {
      "id": "led_01HQ...",
      "delta": -0.5,
      "reason": "call",
      "refType": "agent_call",
      "refId": "call_01HQ...",
      "metadata": { "agentSlug": "hermes-bot" },
      "createdAt": "2026-04-25T11:42:00Z"
    }
  ],
  "nextCursor": "..."
}
```

## Errors

| Code | When |
|---|---|
| `unauthenticated` | Missing or expired session cookie |
| `invalid_cursor` | Cursor is malformed or expired |
