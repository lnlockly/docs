title: "Установка кошелька"
description: "Поставьте MetaMask, сохраните seed-фразу, добавьте BNB Smart Chain. Версия на пять минут."
---

EVM-кошелёк нужен, чтобы пользоваться AgentFlow: войти, оформить подписку, получать заработок, обменивать $FLOW. По умолчанию рекомендуем MetaMask. Альтернативы внизу страницы работают так же.

<Steps titleSize="h3">
  <Step title="Поставьте MetaMask">
    Зайдите на [metamask.io](https://metamask.io) и поставьте расширение для браузера или мобильное приложение.

    <Warning>
      Скачивайте только с официального сайта или из магазина приложений вашей платформы. Поддельные копии встречаются часто.
    </Warning>
  </Step>
  <Step title="Создайте новый кошелёк">
    Откройте MetaMask, нажмите **Create a new wallet**, придумайте надёжный пароль.

    Приложение покажет **seed-фразу из 12 слов**. Запишите её на бумаге и сохраните где-то надёжно — несгораемый сейф, аппаратный бэкап, любое место офлайн.

    <Warning>
      У того, у кого есть seed-фраза, есть кошелёк. Никогда не вводите её на сайтах. Никогда никому не передавайте. Команда AgentFlow никогда её не спросит.
    </Warning>

    <Frame caption="MetaMask показывает seed-фразу из 12 слов">
      ![Экран seed-фразы MetaMask, заглушка](/images/wallet/metamask-seed.png)
    </Frame>
  </Step>
  <Step title="Добавьте BNB Smart Chain">
    AgentFlow работает на BNB Smart Chain (BSC). Добавьте сеть вручную:

    | Поле | Значение |
    |---|---|
    | Имя сети | BNB Smart Chain |
    | RPC URL | `https://bsc-dataseed.binance.org/` |
    | Chain ID | `56` |
    | Символ | `BNB` |
    | Обозреватель | `https://bscscan.com` |

    Адрес контракта USDT BEP20: `0x55d398326f99059fF775485246999027B3197955`.

    <Frame caption="Добавление BNB Smart Chain в настройках сетей MetaMask">
      ![Настройки сети BSC, заглушка](/images/wallet/bsc-network.png)
    </Frame>
  </Step>
  <Step title="Положите немного BNB на газ">
    Каждая транзакция в BSC стоит немного BNB. Купите BNB на любой бирже (Binance, OKX, Bybit и т. п.) и выведите на адрес MetaMask в сети BSC.
  </Step>
</Steps>

## Альтернативы

| Кошелёк | Заметки |
|---|---|
| Rabby | Удобный экран безопасности, симуляция транзакций. Есть расширение и мобильный. |
| Trust Wallet | Мобильный, поддерживает покупку через MoonPay. |
| OKX Wallet | Встроенный DEX-агрегатор, поддержка BSC из коробки. |
| Coinbase Wallet | Дружелюбный для новичков, тянет средства из Coinbase Pay. |

Любой из них работает с AgentFlow. Шаг подключения одинаковый — см. [Подключение кошелька](/ru/wallet/connect).

<Note>
  AgentFlow **не** хранит ваш кошелёк. Мы никогда не видим seed-фразу или приватный ключ. Sign-In With Ethereum (SIWE) только подтверждает, что адрес ваш — деньги при этом не двигаются.
</Note>
