title: "Создать токен"
description: "factory.launch(template_id, params) — развернуть токен через минимальный прокси-клон EIP-1167 дешевле $0.50 в BSC."
---

Создание токена — это вызов `LaunchpadFactory.launch(template_id, params, salt)`. Фабрика клонирует аудированную реализацию запрошенного шаблона, инициализирует её вашими параметрами и возвращает адрес развёрнутого инстанса. Стоимость в BSC — дешевле `$0.50` суммарно.

## Выберите шаблон

| Шаблон | Использовать для | Ссылка |
|---|---|---|
| `virtuals` | Memecoin, токен агента из маркетплейса, всё, что хочет на DEX | [Бондинг-кривая](/ru/concepts/bonding-curve) |
| `dpnm` | Токен закрытой системы со встроенным MLM-деревом, экономика как у `$FLOW` | [Закрытая система](/ru/concepts/closed-system) |

Каждый шаблон принимает свою форму параметров. Кабинет показывает безопасные диапазоны для каждого шаблона; программные запуски через API могут запросить любые значения в рамках платформенных cap.

## Обязательные входы (общие)

<ParamField path="name" type="string" required>
  Отображаемое имя токена, например «Hermes Dispatch».
</ParamField>

<ParamField path="ticker" type="string" required>
  Символ из 3–6 символов, например «HERMES».
</ParamField>

<ParamField path="brief" type="string">
  Опциональное описание проекта одним абзацем. AI заполняет метаданные, картинку, теги из этого.
</ParamField>

<ParamField path="paymentToken" type="address" required>
  ERC20-токен, который используется как pair. В BSC: USDT (`0x55d3...7955`) на mainnet, MockUSDT (`0xD8bD...F04B`) на testnet.
</ParamField>

## Параметры шаблона `virtuals`

<ParamField path="initialPrice" type="number">
  Стартовая цена в `paymentToken` за токен проекта. Дефолт: `0.0001`.
</ParamField>

<ParamField path="graduationThreshold" type="number">
  Заблокированная стоимость `paymentToken`, при которой кривая запечатывается и мигрирует на PancakeSwap V2.
</ParamField>

<ParamField path="failureWindowDays" type="number">
  Дней до того, как не graduated кривая перейдёт в `failed`. Дефолт: `30`.
</ParamField>

## Параметры шаблона `dpnm`

<ParamField path="initialPrice" type="number">
  Стартовая цена в `paymentToken` за токен. `$FLOW` стартовал с `0.1 USDT / FLOW`.
</ParamField>

<ParamField path="dailyLimitFloor" type="number">
  Минимальный дневной лимит покупки на пользователя. Дефолт: `50` (USDT). Эффективный лимит — `max(dailyLimitFloor, pool * dailyLimitRate)`.
</ParamField>

<ParamField path="dailyLimitRate" type="number">
  Доля от пула в дневном лимите. Дефолт: `0.001` (0.1%).
</ParamField>

<ParamField path="incomeLimitMultiplier" type="number">
  Множитель лимита дохода на каждой покупке. Дефолт: `2` (1:2 — купили `$50`, заработаете до `$100`).
</ParamField>

<ParamField path="treeActivationCost" type="number">
  Стоимость активации феноменального дерева на 30 дней в `paymentToken`. Дефолт: `10`.
</ParamField>

## Примеры кода

<CodeGroup>
```typescript ethers.js
import { Contract, Wallet, JsonRpcProvider, parseUnits } from "ethers";
import factoryAbi from "./abi/LaunchpadFactory.json";

const provider = new JsonRpcProvider("https://data-seed-prebsc-1-s1.binance.org:8545");
const wallet = new Wallet(PRIVATE_KEY, provider);
const factory = new Contract(
  "0x713d03A57f9e2c5A526D0C5177342e9417841920",
  factoryAbi,
  wallet,
);

const params = factory.interface.encodeFunctionData("initializeDpnm", [
  "Hermes Token",                // name
  "HERMES",                      // ticker
  "0xD8bDfAAA00496EdE053210f2bcB0702C312FF04B", // MockUSDT
  parseUnits("0.1", 18),         // initialPrice
  parseUnits("50", 18),          // dailyLimitFloor
  10,                            // dailyLimitRate (basis points / 10000)
  2,                             // incomeLimitMultiplier
  parseUnits("10", 18),          // treeActivationCost
]);

const tx = await factory.launch("dpnm", params, ethers.id("hermes-token-v1"));
const receipt = await tx.wait();
const launchEvent = receipt.logs.find(/* Launched(address instance, ...) */);
const instance = launchEvent.args.instance;
console.log("Развёрнут:", instance);
```

```bash REST proxy
curl -X POST https://api.agentflow.website/tokens \
  -H "Content-Type: application/json" \
  -H "Cookie: af_session=..." \
  -d '{
    "template": "dpnm",
    "name": "Hermes Token",
    "ticker": "HERMES",
    "brief": "AI dispatcher token",
    "params": {
      "initialPrice": "0.1",
      "dailyLimitFloor": "50",
      "incomeLimitMultiplier": 2,
      "treeActivationCost": "10"
    }
  }'
```
</CodeGroup>

Кабинет использует RainbowKit + wagmi и отправляет транзакцию прямо с кошелька пользователя. REST-эндпоинт — это удобство для off-chain учёта (метаданные проекта, бриф, картинка), он **не** подписывает за пользователя — ончейн-вызов `factory.launch(...)` всегда инициируется кошельком.

## Ответ

Для REST-прокси:

```json
{
  "id": "tok_01HQ...",
  "slug": "hermes-token",
  "name": "Hermes Token",
  "ticker": "HERMES",
  "template": "dpnm",
  "instance": "0x6B7700395708a0B2578B5E3d58b895C34a24e959",
  "chainId": 97,
  "imageUrl": "https://cdn.agentflow.website/tokens/...",
  "state": "live",
  "createdAt": "2026-04-26T10:31:00Z"
}
```

Токен сразу торгуется. Дальше [прикрепите проект](/ru/launchpad/attach-project), чтобы запустить live-сборку.

<Tip>
  Один кошелёк может запускать несколько токенов. Анти-спам комиссии нет, но токены без прикреплённого проекта быстро падают вниз discovery-ранжирования.
</Tip>

<Note>
  AI-заполнение метаданных (длинное описание, обложка, теги) генерируется off-chain сервисом `tokens-ai-assist` и хранится в БД платформы. Ончейн-вызов фабрики принимает только параметры выше; слой метаданных живёт в `agentflow-api`.
</Note>
