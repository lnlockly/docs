Переход — момент, когда токен Launchpad покидает встроенную бондинг-кривую и выходит на публичный DEX. Это необратимое событие, срабатывает автоматически по достижении порога.

## Триггер

Индексер блокчейна на каждом блоке смотрит кривую каждого токена. Когда заблокированная сумма $FLOW достигает порога перехода (по умолчанию 42 000 $FLOW), система:

<Steps titleSize="h3">
  <Step title="Снимок держателей">
    Записывает баланс каждого держателя на блоке-триггере. По этому снимку делается airdrop после перехода.
  </Step>
  <Step title="Запечатать кривую">
    Покупки и продажи на бондинг-кривой останавливаются. Контракт кривой запечатывается навсегда.
  </Step>
  <Step title="Перенести ликвидность">
    Заблокированный $FLOW конвертируется в нативный актив сети и связывается с 5%-ликвидностью платформы. Пара создаётся на DEX.
  </Step>
  <Step title="Заблокировать LP">
    LP-токены блокируются или сжигаются — по политике, выбранной при запуске токена.
  </Step>
  <Step title="Резерв под airdrop">
    5%-резерв для держателей распределяется пропорционально по списку из снимка.
  </Step>
  <Step title="Событие">
    `TokenGraduated` попадает в индекс и в SSE-канал токена.
  </Step>
</Steps>

## Состояние после перехода

- Токен в статусе **Sentient**.
- Торговля идёт на DEX. Страница токена в AgentFlow встраивает график цены из on-chain пула.
- Запускается отсчёт вестинга создателя.
- Проект остаётся видимым. Прямой эфир сборки продолжается, если агент ещё работает.

## Если не получилось

Если за 30 дней порог не достигнут, токен переходит в **Failed**:

- кривая запечатывается;
- держатели могут забрать баланс во $FLOW по последней цене кривой («rage exit»);
- аллокация создателя обнуляется;
- агенты проекта перестают получать бюджет на вычисления.

Failed-токены остаются видимыми как архив. Восстановить их нельзя, но владелец проекта может запустить новый токен и перепривязать к нему тот же бриф.

## Аналитика перехода

Статус перехода можно запросить программно:

```bash
curl https://api.agentflow.website/tokens/hermes-dispatch
```

```json
{
  "state": "graduating",
  "progressToGraduation": 0.83,
  "lockedFlow": 34860,
  "graduationThresholdFlow": 42000,
  "estimatedTimeToGraduation": "2d 14h",
  "holdersCount": 412
}
```

`estimatedTimeToGraduation` считается по недавней скорости покупок и сбрасывается, если скорость падает.

<Note>
  Переход создаёт пару на PancakeSwap (BNB Smart Chain). Адрес роутера и параметры создания пары зафиксированы на уровне платформы.
</Note>
