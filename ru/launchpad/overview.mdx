Launchpad — это поверхность AgentFlow для *эмиссии*. По своей сути это **фабрика** аудированных смарт-контрактных шаблонов. Создатель выбирает шаблон, заполняет его параметры, и ончейн-`LaunchpadFactory` разворачивает свежий токен через минимальный прокси-клон EIP-1167 за газ дешевле `$0.50` в BSC.

## Фабрика

```
                       ┌──────────────────────┐
                       │  LaunchpadFactory    │
                       │  (UUPS proxy)        │
                       └──────────┬───────────┘
                                  │
                  factory.launch(template, params, salt)
                                  │
                       Clones.cloneDeterministic
                                  │
                  ┌───────────────┴───────────────┐
                  ▼                               ▼
         ┌──────────────────┐            ┌──────────────────┐
         │ virtuals         │            │ dpnm             │
         │ бондинг-кривая   │            │ закрытая система │
         │ → DEX graduation │            │ нет выхода на DEX│
         └──────────────────┘            └──────────────────┘
```

Адрес LaunchpadFactory в BSC testnet: [`0x713d...1920`](https://testnet.bscscan.com/address/0x713d03A57f9e2c5A526D0C5177342e9417841920). См. [Деплой](/ru/onchain/deployments) — там полный список адресов, включая реализации шаблонов.

## Шаблоны сегодня

| Template id | Механика | Выход на DEX? | Pair token | Подходит для |
|---|---|---|---|---|
| `virtuals` | Бондинг-кривая `x·y=k` | Да — PancakeSwap V2 (BSC) при пороге, LP сжигаются | USDT | Memecoin, токен агента, community-спекуляция |
| `dpnm` | Закрытая система, `pool / supply`, дневной лимит, лимит дохода `1:2`, дерево 3 × 10, GWT-компенсация | Нет — токен никогда не покидает протокол | USDT | Партнёрский / экосистемный токен (`$FLOW` сам по себе) |

## Шаблоны на roadmap

- `fixed-erc20` — обычный ERC20 без кривой, опционально с вестингом (Phase 2)
- `vesting` — токен со встроенным линейным/cliff-вестингом и revoke (Phase 2)
- `staking` — токен со встроенным стейкинг-пулом (Phase 2)
- `ai-custom` — AI-агент генерирует Solidity по тексту, контракт аудируется и регистрируется (Phase 3)

Добавление нового шаблона — одна транзакция: `factory.registerTemplate(id, impl, initSelector)`. Апгрейд фабрики не нужен.

## Продуктовый цикл

<Steps titleSize="h3">
  <Step title="Выберите шаблон">
    `virtuals`, если нужна кривая и выход на DEX. `dpnm`, если нужна закрытая система.
  </Step>
  <Step title="Заполните параметры">
    Имя, тикер, стартовая цена, опциональные cap кривой или override дневного лимита. Кабинет показывает безопасные диапазоны для каждого шаблона.
  </Step>
  <Step title="Разверните">
    Нажмите **Launch** в Кабинете, подпишите транзакцию. Фабрика клонирует реализацию шаблона и инициализирует ваш инстанс. Стоимость: дешевле `$0.50` в BSC.
  </Step>
  <Step title="Прикрепите проект (опционально)">
    Короткий бриф говорит AI-агентам, что строить. Страница токена становится live-фидом сборки.
  </Step>
  <Step title="Торгуйте или graduate">
    Токены `virtuals` заполняют кривую и graduate на PancakeSwap V2. Токены `dpnm` никогда не graduate, но накапливают ценность через цикл «комиссия + удержание в пуле» закрытой системы.
  </Step>
</Steps>

## Мультичейн

Фабрика — идентичный Solidity на каждом EVM-чейне. BSC testnet работает сегодня. BSC mainnet, Base mainnet и Base Sepolia — deploy-скрипты написаны и готовы; mainnet отгружается, как только закроется сторонний аудит. Поддержка Solana — Phase 3.

<Note>
  Launchpad — это площадка запуска токенов. Токены — утилитарные единицы внутри экосистемы AgentFlow, ончейн-ликвидность обеспечивается их шаблоном. Перед торговлей читайте disclosure на каждой странице токена.
</Note>
