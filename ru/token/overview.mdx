`$FLOW` — нативный токен платформы. Это настоящий ERC20 в BSC, развёрнутый по адресу [`0x2680...6511`](https://testnet.bscscan.com/address/0x2680b933af7a874d20954b03F38C86891A786511) в testnet через `factory.launch("dpnm", ...)`. Это **не** виртуальный кредит — каждое взаимодействие представляет собой ончейн-транзакцию.

## Главное

| Свойство | Значение |
|---|---|
| Стандарт | ERC20 (BEP20) в BSC |
| Pair token | USDT (MockUSDT в testnet) |
| Ценовая функция | `pool_USDT / total_supply` |
| Обеспечение | 100% USDT в пуле |
| Buy fee | `20%` (10% в феноменальное дерево, 10% остаётся в пуле) |
| Sell fee | `10%` (5% в пул, 5% в treasury) |
| Дневной лимит покупки | `max(50 USDT, pool * 0.001)` на пользователя за 24 часа |
| Лимит дохода | `1:2` на каждой покупке |
| Дерево | 3 ветки × 10 уровней — 88 572 позиций на корень |
| Активация дерева | `10 USDT` на 30 дней, стек до 90 дней |
| Компенсация комиссий | 1:1 в `GWT` (`FlowGrowToken`) |
| Выход на DEX | Нет — закрытая система |

## Точки входа

Все пользовательские взаимодействия идут через контракт `FlowProtocol`. Это единственная точка входа — никогда не вызывайте `$FLOW`, `PhenomenalTree` или `GWT` напрямую.

| Метод | Назначение |
|---|---|
| `activate(referrer)` | Первая регистрация в дереве. Стоит `10 USDT`, даёт 30 дней активного статуса в дереве. |
| `buy(usdtAmount)` | Купить `$FLOW`. Подчиняется дневному лимиту. Добавляет лимит дохода. |
| `sell(flowAmount)` | Продать `$FLOW`. Сжигает лимит дохода пропорционально. |
| `extendTree(months)` | Продлевает активность в дереве на `months × 30` дней. Триггерит маркетинговые выплаты предкам. |
| `buyIncomeLimitWithGWT(gwtAmount)` | Обмен GWT на дополнительный лимит дохода по курсу `1 GWT = $1.25`. |
| `claimGWT()` | Забрать накопленную GWT-компенсацию от прошлых комиссий. |

## Ментальная модель

Каждый `1 USDT`, потраченный на покупку:

```
$0.80 → минт flow-токенов вам ("стоимость на входе")
$0.10 → распределение по предкам в феноменальном дереве как маркетинг
$0.10 → удержание в пуле (это и растит цену)
+
ваш лимит дохода вырастает на $2 (мультипликатор 1:2)
GWT supply +1 GWT начисляется вам (1:1 от комиссии)
```

Каждый `1 USDT` ценности, который вы выводите при продаже:

```
$0.95 → приходит в ваш кошелёк
$0.05 → разбивка: $0.025 удерживается в пуле, $0.025 в treasury
+
ваш лимит дохода сгорает (пропорционально, если value > limit; 1:1, если value <= limit)
GWT supply +0.05 GWT начисляется вам (1:1 от комиссии)
```
