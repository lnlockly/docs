Marketplace — поверхность, где готовые агенты выставляются и вызываются. В отличие от Launchpad, который монетизирует внимание к будущему продукту, Marketplace монетизирует вызовы рабочего продукта.

## Что в листинге

У каждого листинга есть:

- **Persona** (имя, аватар, описание, теги)
- **Цена** (модель и стоимость вызова во FLOW)
- **Статистика** (вызовы, удовлетворённость, p50-латентность)
- **Токен** (опционально — если у агента есть токен Launchpad, он показывается)

## Как покупатели вызывают

Любой залогиненный пользователь может вызвать опубликованного агента. Вызов списывает FLOW: сначала с гранта подписки, затем с пополненного баланса.

Вызов возвращает либо:

- **Синхронный** результат — для коротких агентов
- **SSE-стрим** — для долгих или интерактивных агентов

См. [Цена за вызов](/ru/marketplace/per-call-pricing) для деталей по стоимости.

## Как создатели зарабатывают

Каждый вызов рассчитывается по детерминированной схеме. Для LLM-доли вызова:

| Получатель | Доля |
|---|---|
| Caller (как rebate-кредит) | 50% |
| Создатель | 20% |
| Платформа | 20% |
| Держатели токена | 5% |
| Резерв | 5% |

Для tool-premium-доли (image gen, кастомный HTTP, платные API-интеграции):

| Получатель | Доля |
|---|---|
| Создатель | 80% |
| Платформа | 10% |
| Держатели токена | 10% |

См. [Распределение доходов](/ru/marketplace/earnings-split) — как считаются корзины и краевые случаи (возвраты, неудачные вызовы).
