## 2026-04-26

### `$FLOW` развёрнут через фабрику в BSC testnet

Токен платформы запущен как инстанс шаблона `dpnm`. Каждый
будущий токен в стиле dPNM будет разворачиваться так же — одним
вызовом `factory.launch("dpnm", params)`.

- **Токен $FLOW:** [`0x2680b933af7a874d20954b03F38C86891A786511`](https://testnet.bscscan.com/address/0x2680b933af7a874d20954b03F38C86891A786511)
- **FlowProtocol** (точка входа для `activate`/`buy`/`sell`/`extendTree`): [`0x7F20f8e6e5077a4E11A1A2Fe54449ab55D32f653`](https://testnet.bscscan.com/address/0x7F20f8e6e5077a4E11A1A2Fe54449ab55D32f653)
- Полный список адресов → [Деплой](/ru/onchain/deployments)

### LaunchpadFactory + реестр шаблонов

Единый паттерн фабрики заменил отдельные deploy-скрипты для каждого
токена. Зарегистрировано два шаблона:

- `virtuals` — бондинг-кривая `x·y=k` → graduation на DEX (для мемкоинов и токенов агентов)
- `dpnm` — закрытая система, 100% обеспечение, дневной лимит, лимит дохода, феноменальное дерево 3 × 10 (для `$FLOW` и любого другого MLM-токена)

Архитектура позволяет сторонние клоны — кто угодно может запустить
свой dpnm-токен, не написав ни строчки Solidity. Стоимость в BSC: ~`$0.50`
суммарно на токен через минимальные прокси-клоны EIP-1167.

**Тесты:** 35/35 проходят по фабрике и Flow.

### Документы для аудита смарт-контрактов

Threat model, 30 инвариантов, 15 сценариев атак, скоуп, runbook
воспроизведения и чек-лист соответствия по каждому контракту для
`$FLOW` и dPNM. Хранятся в приватном репозитории контрактов в
`agentflow-contracts/audit-flow/`.

Запланированный путь аудита: **Spearbit private** (`$120k–180k`) → **Code4rena public**
(`$80k–120k` + `$20k` судейство) → **Immunefi bounty** (пул `$500k`, 90 дней
после запуска).

### Чейн-индексер на бэкенде

`agentflow-api` теперь запускает per-chain viem-поллер, который слушает
`Buy`, `Sell`, `Graduated`, `ReferralRewardAccrued` и пишет их в
Postgres (`swap_events`, `referral_accruals`). По умолчанию выключен —
переключите `CHAIN_INDEXER_ENABLED=1` в env, как только адреса фабрики готовы.

Миграция `0014_chain_indexer.sql` расширяет `tokens` метаданными чейна
и добавляет таблицу состояния индексера.

## Раньше

### Marketplace v3 + детальная страница агента v3

`/marketplace` теперь начинается с featured-ряда (топ-6 агентов по
объёму вызовов) плюс 4-колоночная сетка всех живых агентов. Никаких
фейковых mock-карточек. Детальная страница переписана в стиле REPPO с
табами (Overview / Capabilities / Earnings / Reviews) и липкой правой
панелью «Try this agent».

### Live build view (в духе v0.dev)

`/live/:slug` — это 3-панельный workspace: сайдбар проекта + табовая
контентная зона (**Preview** / Code / Terminal / Network). Панель
Preview подстраивается под `kind` проекта — веб-приложение получает
iframe с дашбордом, account farm — холст с 100-точечной сеткой
здоровья, парсер — реал-тайм бар-чарт и так далее.

### UI лаунчпада токенов (мультичейн)

`/launch` и `/launch/:slug` идут с селектором сети (BSC / Base),
AI-помощником, который из одного предложения брифа генерирует тикер,
имя, описание и логотип, потоком транзакции через RainbowKit и модалкой
прогресса в 4 шага.

### AI Guide assistant

Сайдбар в стиле Cloudflare, который проводит пользователя по
платформе. Tool-use API: `navigate`, `highlight`, `scroll_to`, `fill`
(с подтверждением), `start_tour`, `open_external`. Десктоп — правая
панель, мобайл — bottom-sheet, Telegram MiniApp с нативной BackButton.

### Голосовой режим во всех чатах

Кнопка микрофона press-and-hold плюс опт-ин TTS-тоггл в ChatInline и
GuidePanel. Web Speech API там, где поддерживается, fallback через
Whisper-прокси на iOS Safari и Firefox.

### FLOW credits + выплаты

Подписки (Plus / Pro / Max → 20 / 100 / 200 FLOW ежемесячно),
CryptoBot + Platega + BSC USDT BEP20 как onramp, разделение
маркетплейсных earnings (50 / 20 / 20 / 5 / 5 для LLM-бакета;
80 / 10 / 10 для tool-бакета), флоу вывода средств для создателей с
ручным апрувом админа.
