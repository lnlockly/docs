title: "Подписки"

---

Подписки — способ получить FLOW со скидкой и разблокировать приоритетные функции. Три тарифа, единый поток биллинга.

## Тарифы

| Тариф | Месяц USD | Месячный грант FLOW | Возможности |
|---|---|---|---|
| **Plus** | $20 | 20 FLOW | Приоритетная очередь, базовая аналитика |
| **Pro** | $100 | 100 FLOW | Всё из Plus + premium-агенты, расширенная аналитика, ускоренный SSE |
| **Max** | $200 | 200 FLOW | Всё из Pro + приватные страницы проектов, выделенный слот рантайма |

Грант начисляется при активации и повторно — каждые 30 дней через cron `subscription-grants`. Неизрасходованный грант FLOW не переносится и сбрасывается при следующем гранте.

## Активация подписки

```bash
curl -X POST https://api.agentflow.website/subscriptions/checkout \
  -H "Content-Type: application/json" \
  -H "Cookie: af_session=..." \
  -d '{
    "tier": "pro",
    "provider": "cryptobot"
  }'
```

<ParamField path="tier" type="string" required>
  Одно из `plus`, `pro`, `max`.
</ParamField>

<ParamField path="provider" type="string" required>
  Одно из `cryptobot`, `platega`, `usdt_bep20`.
</ParamField>

В ответе придёт `checkoutUrl` для прохода через провайдера. Перенаправьте пользователя на этот URL.

```json
{
  "subscriptionId": "sub_01HQ...",
  "tier": "pro",
  "provider": "cryptobot",
  "checkoutUrl": "https://t.me/CryptoBot?start=invoice-...",
  "expiresAt": "2026-04-25T11:30:00Z"
}
```

После того как пользователь оплачивает, вебхук провайдера активирует подписку и зачисляет первый грант.

## Авто-продление

Подписки авто-продлеваются через того же провайдера, что и при активации. Если оплата продления не прошла, подписка переходит в 7-дневный grace-период. Во время grace:

- Имеющийся грант FLOW можно тратить.
- Новый грант не начисляется.
- Платёж пробуется ежедневно.

После 7 дней неудачных попыток подписка отменяется. Пользователь может подписаться снова в любой момент.

## Отмена

Пользователь отменяет подписку через Cabinet. Отмена вступает в силу в конце периода — пользователь сохраняет текущий грант до истечения, затем падает обратно на pay-as-you-go.

## Моя подписка

```bash
curl https://api.agentflow.website/me/subscription \
  -H "Cookie: af_session=..."
```

```json
{
  "tier": "pro",
  "status": "active",
  "monthlyGrantFlow": 100,
  "currentGrantRemainingFlow": 73.4,
  "renewsAt": "2026-05-25T11:30:00Z",
  "provider": "cryptobot"
}
```

<Warning>
  Понижение тарифа вступает в силу при следующем продлении. Повышение — мгновенно, с pro-rata-перерасчётом неиспользованного остатка текущего периода.
</Warning>
