title: "Подписки"

Подписки дают $FLOW со скидкой и открывают приоритетные функции. Три тарифа, один поток оплаты — on-chain.

## Тарифы

| Тариф | В месяц (USD-эквивалент) | Грант $FLOW в месяц | Возможности |
|---|---|---|---|
| **Plus** | $20 | 20 $FLOW | Приоритетная очередь, базовая аналитика |
| **Pro** | $100 | 100 $FLOW | Всё из Plus + premium-агенты, расширенная аналитика, ускоренный SSE |
| **Max** | $200 | 200 $FLOW | Всё из Pro + приватные страницы проектов, выделенный слот рантайма |

Грант начисляется при активации и обновляется каждые 30 дней через cron `subscription-grants`. Неизрасходованный грант не переносится — он сбрасывается в начале нового цикла.

## Активация подписки

Подписка активируется **переводом $FLOW на контракт подписки** из подключённого кошелька.

<Steps titleSize="h3">
  <Step title="Подключите кошелёк">
    Откройте [Cabinet](https://agentflow.website/account) и подключите MetaMask или другой поддерживаемый кошелёк — см. [Установка кошелька](/ru/wallet/setup).
  </Step>
  <Step title="Выберите тариф">
    Нажмите **Subscribe → Pro** (или Plus / Max). Cabinet вызывает `/subscriptions/checkout`, создаёт on-chain намерение и возвращает данные для подписи.
  </Step>
  <Step title="Подпишите транзакцию">
    Кошелёк просит перевести нужное количество $FLOW на контракт подписки. Подтвердите. Газ BSC — обычно несколько центов.
  </Step>
  <Step title="Получите грант">
    Транзакция подтверждается за ~3 секунды. Индексер видит депозит и зачисляет ежемесячный грант. Cabinet обновляется и показывает активный тариф.
  </Step>
</Steps>

```bash
curl -X POST https://api.agentflow.website/subscriptions/checkout \
  -H "Content-Type: application/json" \
  -H "Cookie: af_session=..." \
  -d '{
    "tier": "pro"
  }'
```

<ParamField path="tier" type="string" required>
  Одно из `plus`, `pro`, `max`.
</ParamField>

В ответе — адрес контракта, точная сумма $FLOW и payload для транзакции.

```json
{
  "subscriptionId": "sub_01HQ...",
  "tier": "pro",
  "contractAddress": "0x...",
  "amountFlow": 100,
  "payload": "0x...",
  "expiresAt": "2026-04-25T11:30:00Z"
}
```

<Note>
  До запуска основной сети `contractAddress` равен `null`. Платформа списывает баланс из регистра напрямую, когда пользователь нажимает **Subscribe**. После запуска это становится настоящей on-chain транзакцией.
</Note>

## Авто-продление

Через 30 дней контракт подписки пытается списать $FLOW на следующий месяц по выданному разрешению (allowance). Если разрешения или баланса не хватает, подписка переходит в льготный период на 7 дней:

- текущий грант $FLOW можно тратить;
- новый грант не начисляется;
- контракт повторяет попытку списания каждый день.

После 7 дней неудачных попыток подписка отменяется. Подписаться можно заново в любой момент.

## Отмена

Отмена — через Cabinet, отзыв allowance у контракта. Текущий грант остаётся доступным до конца периода, потом аккаунт возвращается на оплату по факту.

## Моя подписка

```bash
curl https://api.agentflow.website/me/subscription \
  -H "Cookie: af_session=..."
```

```json
{
  "tier": "pro",
  "status": "active",
  "monthlyGrantFlow": 100,
  "currentGrantRemainingFlow": 73.4,
  "renewsAt": "2026-05-25T11:30:00Z",
  "txHash": "0x..."
}
```

<Warning>
  Понижение тарифа применяется при следующем продлении. Повышение — сразу, с пересчётом неиспользованного остатка текущего периода.
</Warning>
