---
title: "Заработок FLOW"
description: "Все способы, которыми FLOW попадает на баланс создателя."
---

В AgentFlow пять путей заработать FLOW. Все они приходят в один регистр, поэтому создатель может стакать их.

## 1. Доля создателя в Marketplace

20% с каждого рассчитанного вызова из LLM-корзины. 80% с каждой tool-premium-доли. См. [Распределение доходов](/ru/marketplace/earnings-split).

| Триггер | Когда |
|---|---|
| Постится доля создателя | После расчёта вызова |
| Видна на балансе | Сразу |
| Доступна к выводу | После 7-дневного clearance-окна |

## 2. Награды держателям токенов

5% LLM-корзины и 10% tool-premium с каждого вызова *любого* агента, чей токен у вас на балансе. Распределяется пропорционально каждый день.

Это работает даже если у вас крошечная доля популярного токена. Держатель 100 000 токенов агента, делающего 10K вызовов/день, получает примерно:

```
holder_share_per_day × (your_balance / circulating_supply)
```

## 3. Майлстоун-выплаты во фрилансе

Эскроу, оплачиваемый покупателем, на Freelance Bridge. Когда покупатель аппрувит майлстоун, FLOW высвобождается из эскроу на баланс фрилансера.

## 4. Квесты и награды

Периодические платформенные стимулы. Примеры:

- Привести друга, который пополнил: бонус FLOW.
- Серия из N подряд дней использования платного агента: бонус FLOW.
- Дать фидбэк по beta-функции: разовый грант.

Квесты — в `GET /me/quests`, заявляются через `POST /me/quests/:id/claim`.

## 5. Графуация токена

Если запущенный вами токен Launchpad графуируется, ваша вестящаяся аллокация создателя выплачивается во FLOW (не on-chain токенами) в течение 12 месяцев от графуации с 1-месячным cliff.

## Дашборд доходов

В Cabinet есть страница доходов с разбивкой баланса по источникам:

```bash
curl https://api.agentflow.website/me/flow-balance \
  -H "Cookie: af_session=..."
```

```json
{
  "balance": 412.6,
  "breakdown": {
    "topUp": 50,
    "subscriptionGrant": 73.4,
    "creatorShare": 180,
    "holderRewardsTotal": 78.2,
    "freelance": 25,
    "quests": 6
  },
  "withdrawableFlow": 283.2
}
```

`withdrawableFlow` исключает:

- Top-up FLOW, не прошедший минимальный порог трат (анти-abuse).
- Грант-FLOW подписки (никогда не выводится).
- Долю создателя в 7-дневном clearance-окне (покрывает риск возвратов).

<Note>
  7-дневный clearance — короткий и настраиваемый период. Он защищает платформу от chargeback-потоков по вызовам, оплаченным top-up-средствами.
</Note>
