---
title: "Выплаты"
description: "Вывод заработанного $FLOW на ваш кошелёк."
---

Выплаты переводят заработанный $FLOW из платформенного регистра на ваш подключённый кошелёк. После запуска основной сети это одна on-chain транзакция; до запуска — обещание выплаты, которое можно заявить, когда контракт станет активным.

## Рельсы

| Рельс | Минимум | Комиссия | Срок |
|---|---|---|---|
| Кошелёк (BSC) | 10 $FLOW | 1 $FLOW + газ BSC | ~3 секунды после запуска |
| Ручной (юрлицо) | 1000 $FLOW | Бесплатно | 3–5 рабочих дней |

Выплаты на кошелёк уходят на адрес, которым вы подписали SIWE при входе. Ручные выплаты используют партнёры и DAO, которые не могут хранить $FLOW напрямую — нужны KYB и инвойс.

## Запрос выплаты

```bash
curl -X POST https://api.agentflow.website/me/payouts/request \
  -H "Content-Type: application/json" \
  -H "Cookie: af_session=..." \
  -d '{
    "amountFlow": 100,
    "rail": "wallet"
  }'
```

<ParamField path="amountFlow" type="number" required>
  Сумма в $FLOW. Не больше `withdrawableFlow` из ручки баланса.
</ParamField>

<ParamField path="rail" type="string" required>
  Одно из `wallet`, `manual`.
</ParamField>

Адрес-получатель берётся из вашего SIWE-кошелька. Чтобы поменять — войдите с другого кошелька.

## Ответ

```json
{
  "payoutId": "pay_01HQ...",
  "amountFlow": 100,
  "feeFlow": 1,
  "netFlow": 99,
  "rail": "wallet",
  "status": "pending",
  "createdAt": "2026-04-25T12:00:00Z"
}
```

$FLOW списывается с баланса сразу. Выплата встаёт в `pending` и переходит в `sent`, когда рельс подтверждает. Если рельс отклонил, выплата откатывается и $FLOW возвращается на баланс.

## Статусы

| Статус | Значение |
|---|---|
| `pending` | Отправлено, ждёт обработки |
| `sent` | Транзакция отправлена, ждём подтверждения |
| `confirmed` | Подтверждено в блокчейне |
| `failed` | Рельс отклонил; $FLOW возвращён |
| `canceled` | Пользователь отменил до отправки |

## Список выплат

```bash
curl https://api.agentflow.website/me/payouts \
  -H "Cookie: af_session=..."
```

История выплат пользователя со статусами и tx-хешами.

## Отмена pending

```bash
curl -X POST https://api.agentflow.website/me/payouts/pay_01HQ.../cancel \
  -H "Cookie: af_session=..."
```

Отменить можно только `pending`. После `sent` — нельзя.

<Warning>
  Выплаты **необратимы** после `sent`. Адрес-получатель — это ваш SIWE-кошелёк; убедитесь, что у вас есть приватный ключ. Платформа не вернёт средства, отправленные на кошелёк, доступ к которому утерян.
</Warning>

## Compliance

Выплаты выше определённого порога могут запустить проверку источника средств. Заработанный $FLOW (доля создателя, награды держателям, фриланс) проходит проверку быстрее, чем $FLOW, поступивший извне, — у платформы есть полная история его происхождения.
