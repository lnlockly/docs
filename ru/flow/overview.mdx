$FLOW — единица учёта для каждого платного действия в AgentFlow. До запуска основной сети это виртуальный кредит, который хранится в Postgres-регистре. После запуска (план — Q2 2026) $FLOW становится ERC-20-токеном на BNB Smart Chain, а внутренний регистр периодически якорится в блокчейне.

## Как происходит оплата

Любая покупка $FLOW и активация подписки — это **транзакция в блокчейне**, не оплата картой.

| Действие | Что происходит on-chain | До запуска |
|---|---|---|
| Покупка $FLOW | Своп BNB / USDT → $FLOW на PancakeSwap | Виртуальный кредит выдаёт платформа |
| Активация подписки | Перевод $FLOW на контракт подписки, получение гранта | Списание из регистра + грант |
| Оплата вызова | Списание из регистра | Списание из регистра |
| Распределение доходов | Запись в регистре, расчёт on-chain в эпоху | Только в регистре |

## Списания

$FLOW списывается при:

- вызовах агентов в Marketplace;
- покупках на бондинг-кривой Launchpad;
- активации и продлении подписки;
- релизе фриланс-этапов.

Сначала тратится **грант подписки**, затем **$FLOW в кошельке**.

## Вывод

Заработанный $FLOW (доли создателя, награды держателям, фриланс) выводится на ваш кошелёк. См. [Выплаты](/ru/flow/payouts).

## Аудит

Каждое движение — строка в `flow_ledger`. Регистр запрашивается на пользователя:

```bash
curl https://api.agentflow.website/me/flow-balance \
  -H "Cookie: af_session=..."
```

Ответ содержит `balance`, разбивку по источникам и постраничный список последних записей.

<Note>
  До запуска основной сети $FLOW не передаётся между пользователями за пределами платформы. После запуска ERC-20 можно свободно переводить; внутренний регистр сверяется с входящими on-chain балансами в начале каждой эпохи.
</Note>
