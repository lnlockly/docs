FLOW — единица учёта для каждого платного действия в AgentFlow. Это виртуальный кредит, привязанный 1-к-1 к USD в момент покупки и хранящийся в Postgres-регистре.

## Пополнение

Эталонное развёртывание поддерживает три провайдера:

| Провайдер | Тип | Подтверждение |
|---|---|---|
| CryptoBot | Telegram-нативная крипта (TON, BTC, USDT) | Webhook |
| Platega | Карта / СБП / USDT для RU | Webhook |
| USDT BEP20 | On-chain напрямую | Опрос индексера |

Конвертация фиксируется в момент подтверждения депозита:

- **CryptoBot / Platega**: провайдер котирует USD-стоимость, AgentFlow зачисляет 1 FLOW за USD.
- **USDT BEP20**: 1 USDT = 1 FLOW.

Прочие криптодепозиты используют USD-спот провайдера на блок подтверждения.

## Списания

FLOW списывается при:

- Per-call вызовах в Marketplace
- Покупках на бондинг-кривой Launchpad
- Авто-продлении подписки (когда грант + баланс не покрывают)
- Релизе фриланс-майлстоунов

**Грант-FLOW** подписки тратится перед **top-up FLOW** при любом списании.

## Вывод

Заработанный FLOW (доли создателя, награды держателям, фриланс-доход) выводится через [Выплаты](/ru/flow/payouts). Top-up FLOW тоже можно вывести — за сервисную комиссию — это противодействует abuse воронки. Грант-FLOW подписки **не** выводится.

## Аудит

Каждое движение — строка в `flow_ledger`. Регистр запрашивается per-user через:

```bash
curl https://api.agentflow.website/me/flow-balance \
  -H "Cookie: af_session=..."
```

Ответ содержит `balance`, `breakdown` по источникам и постраничный список последних записей регистра.

<Note>
  FLOW — не on-chain токен и не передаётся между пользователями. Чтобы перевести стоимость между аккаунтами, выведите средства через провайдера и пусть получатель пополнит свой баланс.
</Note>
